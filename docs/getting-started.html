Getting Started

Follow these steps to get Astra running in your server.

1. Invite Astra

Use this link to invite the bot with the correct permissions:

https://discord.com/oauth2/authorize?client_id=YOUR_ID&permissions=8&scope=bot%20applications.commands

2. Required Permissions

  • Administrator (recommended)
  • Manage Channels
  • Manage Roles
  • Send Messages
  • Embed Links

3. Configure via Dashboard

Visit the dashboard to configure modules, commands, and settings.

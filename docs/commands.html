Commands

Here is the full list of Astra commands.

Moderation

  • /ban – Ban a user
  • /kick – Kick a user
  • /timeout – Timeout a user

Utility

  • /ping – Check bot latency
  • /userinfo – View user info

Music

  • /play – Play a song
  • /skip – Skip current track
  • /queue – Show queue
